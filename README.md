# mentorship-sessions-nandaa

Mentoship sessions by Nandaa. 

## Submitting Assignments

To submit assignments/challenges:

1. Fork this repo to your Github account
1. Create a folder/directory with your Github username under `/assignments`, eg. if your username is `octcat`, put all your assignments under `/assignments/octcat`. If you have multiple files for a single assignment, you can put them under a child folder/directory.
1. Once done with the assignments, do the Pull Request.
